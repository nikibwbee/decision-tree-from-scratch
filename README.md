# Decision Tree From Scratch

This project implements a **Decision Tree classifier from scratch** using **Python and NumPy**, without scikit-learn.

## Dataset
- `tennis_tiny.csv` contains a small dataset to predict `PlayTennis` based on weather conditions.

## Features
- Outlook (Sunny, Overcast, Rain)
- Temperature (Hot, Mild, Cool)
- Humidity (High, Normal)
- Windy (True/False)
